Problema "matriz_geral"

Ler uma matriz quadrada de ordem N (máximo = 10), contendo números reais. Em seguida, fazer as seguintes ações:

  1. Calcular e imprimir a soma de todos os elementos positivos da matriz.
  2. Fazer a leitura do índice de uma linha da matriz e, daí, imprimir todos os elementos desta linha.
  3. Fazer a leitura do índice de uma coluna da matriz e, daí, imprimir todos os elementos desta coluna.
  4. Imprimir os elementos da diagonal principal da matriz.
  5. Alterar a matriz elevando ao quadrado todos os números negativos da mesma. Em seguida imprimir a matriz alterada.
